# HAI2026 Interactive Slides

Interactive tutorial slides for building a data analysis tool with OpenAI API and Streamlit.

## Live Demo

https://bchaicourse.github.io/HAI2026-Interactive-Slides/

## Local Development

```bash
cd interactive-slides
npm install
npm run dev
```

## Build for Production

```bash
cd interactive-slides
npm run build
```

Built files will be in the `docs/` folder for GitHub Pages deployment.
